# 🏆 Fantacaserma — Asta a Busta Chiusa

Web app per gestire l'asta live a busta chiusa del fantacalcio.

## Deploy su Render (gratuito)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/darioschioppi/fantacaserma)

1. Clicca il pulsante sopra
2. Accedi con GitHub
3. Clicca "Apply" → Deploy automatico

## Come funziona

1. Admin seleziona un giocatore e avvia l'asta
2. Tutte le squadre hanno 30 secondi per inviare l'offerta
3. Il tavolo poker mostra chi ha offerto (senza importo)
4. Alla scadenza: le buste si aprono
5. In caso di parità → spareggio inline nella stessa schermata
